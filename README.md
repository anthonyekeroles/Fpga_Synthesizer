FPGA Audio Synthesizer (Nexys A7)

This project implements a simple digital audio synthesizer on the Nexys A7 FPGA. Using onboard switches as inputs, the system generates real-time audio tones corresponding to musical notes across multiple octaves.

Square waves are generated by toggling signals at frequencies derived from note-specific half-period calculations, while sine waves are implemented using a lookup table (LUT) for smoother sound output. The design supports polyphony by combining multiple note signals, allowing chords to be played simultaneously.

The project demonstrates key digital design concepts including frequency division, waveform generation, lookup tables, and real-time signal processing in Verilog.
